# kubeadm Commands

## Initialize Control Plane

Executed on the master node:

```bash
sudo kubeadm init \
  --apiserver-advertise-address=172.31.12.17 \
  --pod-network-cidr=192.168.0.0/16
```

## Configure kubectl

```bash
mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config
```

## Install Calico CNI

```bash
curl -O https://raw.githubusercontent.com/projectcalico/calico/v3.32.0/manifests/calico.yaml
kubectl apply -f calico.yaml
```

## Join Worker Nodes

The worker nodes were joined using the command generated by kubeadm.

The actual command is not included because it contains a temporary token.

General format:

```bash
sudo kubeadm join <MASTER_PRIVATE_IP>:6443 \
  --token <TOKEN> \
  --discovery-token-ca-cert-hash sha256:<HASH>
```

## Verify Cluster

```bash
kubectl get nodes
kubectl get nodes -o wide
kubectl get pods -A -o wide
```
